# 🛒 Brazilian E-Commerce SQL Analytics

## 📌 Project Overview

This project demonstrates SQL-based data analysis using the Brazilian E-Commerce dataset. It focuses on extracting meaningful business insights through SQL queries related to sales, customers, products, orders, revenue, delivery performance, and seller analysis.

The project showcases practical SQL skills used in real-world business intelligence and data analytics.

---

## 🎯 Project Objectives

- Analyze overall order performance.
- Calculate total revenue.
- Identify top-selling product categories.
- Analyze customer purchasing behavior.
- Evaluate state-wise order distribution.
- Measure average order value.
- Track monthly order trends.
- Analyze delivery performance.
- Identify top-performing sellers.
- Generate business insights using SQL.

---

## 📂 Database Tables

The project uses the following tables:
